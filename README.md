# 로아 떠상 (Wandering Merchant)

로스트아크 떠돌이 상인에서 **전설 카드**가 등장했을 때 알림을 주는 Chrome 확장프로그램입니다.

## 기능

- 전설 카드 등장 시 브라우저 알림
- 현재 나와있는 전설 카드 목록 표시
- 서버 선택 기능

## 설치

[Chrome 웹 스토어](https://chrome.google.com/webstore)에서 "로아 떠상"을 검색하거나 직접 설치할 수 있습니다.

## 로컬 개발 실행 방법

1. 로컬 서버 실행:

   ```bash
   python -m http.server 8000 --bind 127.0.0.1
   ```

2. 웹 브라우저에서 `http://127.0.0.1:8000` 접속
3. 코드 수정 후 F5로 새로고침
